Question 1: Explain the difference between an Array and a Linked List regarding time complexity of insertions.

An array requires shifting elements when inserting in the middle or beginning, resulting in O(N) time complexity. However, inserting at the end is O(1) amortized if there is space.

A linked list allows O(1) insertions anywhere IF you already have a reference to the node you are inserting after. However, finding that spot is O(N).


Question 2: Write a method to reverse a singly linked list.


public Node reverse(Node head) {
    Node prev = null;
    Node current = head;
    Node next = null;
    while (current != null) {
        next = current.next;
        current.next = prev;
        prev = current;
        current = next;
    }
    return prev;
}
